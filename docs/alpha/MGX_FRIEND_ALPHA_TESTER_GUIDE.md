# MGX Friend Alpha Tester Guide

Date: 2026-05-16

## URL

`https://mgx-poker.com/`

## Games To Try

Please focus on these five games:

| Game | Menu Label |
| --- | --- |
| Badugi | Badugi |
| 2-7 Triple Draw | 2-7 Triple Draw / D01 |
| A-5 Triple Draw | A-5 Triple Draw / D02 |
| 2-7 Single Draw | 2-7 Single Draw / S01 |
| A-5 Single Draw | A-5 Single Draw / S02 |

## Games To Avoid

Do not enable preview variants. Board games, Omaha, Stud/Razz, Dramaha, and Chinese/OFC are not part of this friend alpha.

## What To Check

- Does each game start normally?
- On Badugi, does the game avoid freezing through Draw 1, Draw 2, Draw 3, result, and next hand?
- Does the pot stay visible and non-confusing during a hand?
- Does the acting order feel obviously wrong?
- Are action buttons visible and tappable on mobile?
- Is Fold visible when it is available?
- Are cards, player panels, pot, and result overlay readable?
- Does next hand work after the result?
- Does anything disappear, overlap, or get stuck?

## How To Report Bugs

Please send:

1. Game name.
2. Phone / browser / orientation.
3. What you tapped immediately before the issue.
4. Screenshot or screen recording if possible.
5. Whether refresh fixed it.

## Known Limitations

- This is a closed alpha.
- UI and AI behavior are still being tuned.
- Some non-Core games are intentionally disabled or preview-only.
- Badugi is included because it is a core MGX game, but pot/terminal/action behavior should be watched closely.
